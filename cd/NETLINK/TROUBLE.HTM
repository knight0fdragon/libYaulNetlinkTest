Troubleshooting


Every place I try says that the server can't be found. Check your DNS settings in the Configuration Options.


I have voice mail but I can't get connected. If your voice mail system changes the dial-tone (by beeping, for example), the modem may have trouble recognizing the dial tone. If this is the case, use the "Blind Dialing" option.


I got a -404 error. This error indicates that the requested page is not available on the server. It is possible that the page has moved. You might try starting at the first document on the server by clicking on the document URL area and deleting everything after the third "/".


I type on the keyboard and it does not work. Disconnect keyboard adapter and reinsert into Sega Saturn controller, press port. Wait for 3 seconds. You should be able to type again. Do not press any keys on the keyboard when you push the Sega Saturn Reset button or you will need to redo this procedure.


My download stopped in the middle. Sometimes the data is interrupted in its journey over the internet. Click "Stop"" and then click "Reload." This will start the download over again. You can also click "Back" to go to the previous screen.


I got a 503 error. This indicates that the server is too busy to send the page. Try again in a few minutes.


The page didn't appear to finish loading. If the page was large or contained lots of graphics, it is possible the Browser ran out of memory trying to display it. Try turning the picture loading off by clicking "Images" in the toolbar.


I went to a site that asked for a password. If you go to a web page that requires a membership password and you don't have the password, press "Start" to exit screen.


If you still have questions and are able to connect to the Internet, try the Net Link home page where you can review the Frequently Asked Question (FAQ) listings to see if your question is answered there.
