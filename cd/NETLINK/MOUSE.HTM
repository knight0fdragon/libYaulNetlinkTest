Using The Optional Net Link Mouse

Roll the Mouse over a flat surface to move the Arrow in any direction. (If you want to view below the current screen, move the Mouse so that the Arrow touches the bottom of the screen. Move the Arrow away from the bottom to stop the screen from scrolling. Touch the Arrow to the top of the screen to display what appears above the current screen. Move the Arrow away from the top to stop the screen from scrolling.)

  • Left Button Press any time you want to activate an icon (perform a function or action). Works like the A Button on the Sega Saturn Controller.

  • Middle Button Brings up your list of Bookmarks and saved sites. (You can use your Left Button to choose a bookmarked site to visit, or press the START Button to exit Bookmarks without choosing a destination.)

  • Right Button Magnifies the area under your Arrow. Acts the same as the C Button on the Sega Saturn Controller.

  • START Button Brings up the Command Cluster.


Using the Optional Net Link Keyboard Adapter (with a Computer Keyboard)


The Net Link Keyboard Adapter allows you to attach any standard small plug (mini-DIN) PC keyboard to your Sega Saturn. To install, plug the Keyboard Adapter into your Sega Saturn right controller port and then plug your PC keyboard into the connector. You can use either a 101-key keyboard or an 84-key keyboard.


If your PC keyboard does not have a small plug (mini-DIN) PC compatible connector, you will need to purchase a special converter. You can purchase this adapter, if needed, from most major computer retailers.


Once you've connected the keyboard and adapter you can type as you normally would on a computer. If you are using an 84-key keyboard, however, you will not have access to some of the special "quick" keys listed below.


Use the keyboard to type character, change or enter a URL, write and send e-mail and enter searches on the Web. Some keys on the keyboard can act as special Quick-Keys that perform an action quickly. The following is a list of the Quick-Keys you can use:

  • ENTER Takes you to the Web page designated by the URL bar.

  • ESC Deletes the contents of the selected entry.

  • F1-F3 The same as the three buttons on the right of the on-screen keyboard. (Using the Shift key with any of these keys affects them the same as the on-screen keyboard.)

  • F4 Same as the START key. Causes the on-screen keyboard to be put away.


If you are using the keyboard adapter and notice that your Sega Saturn is not accepting input from the keyboard, do not be alarmed. Simply unplug the keyboard adapter from your Sega Saturn, wait a few seconds and plug it back in. Wait a few more seconds. Do not press the "Reset" button on your Sega Saturn while holding down any key on a keyboard attached to the keyboard adapter. This will prevent your Sega Saturn from being able to access your keyboard. Again, if your Sega Saturn has problens accessing the keyboard, simply unplug your keyboard adapter from your Sega Saturn, wait a few seconds and plug it back in and wait a few more seconds. You should then be able to type normally.
