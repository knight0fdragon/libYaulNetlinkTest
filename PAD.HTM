Using The Saturn Control Pad

The following are button-by-button instructions for operating the Net Link web browser's interface.

  • D-Pad Controls the positioning of the Arrow. (If you want to scroll the page, press the D-Pad DOWN so that the on-screen Arrow touches the bottom of the screen. Keep pressing down and the entire page will scroll up so you can see what's below. Press the D-Pad UP and hold to move the Arrow to the top of the screen and scroll the page down.) You can also press the D-Pad LEFT and RIGHT to scroll the screen left and right.

  • A Button Perform an Action on any icon or Link. (Highlighted text signifies a Link to something specific on the Web. Press the A Button while the Arrow is over a Link and you will go to that location. Press the A Button while you're pointing at a Toolbar Icon to choose that Icon's Action. Press the A Button while pointing at the URL bar at the top of the screen and you will be able to change the URL.)

  • B Button Brings up your list of Bookmarks or saved sites. (Use your A Button to choose a bookmarked site to visit, or press the START Button to exit Bookmarks without choosing a destination.)

  • C Button Magnifies the area under your Arrow. (Press the C Button once to bring up the magnifier. You can position the magnifier over anything on screen to enlarge and make it easier to see. Press the C Button again to make the magnifier larger and one more time for the maximum size. A fourth push of the C Button will put the magnifier away and return you to the normal cursor.)

  • START Button Brings up the Command Cluster. (The Command Cluster is a quick way to access the most commonly used Toolbar items. To choose an icon from the Command Cluster, press the START Button and use the D-Pad to select an Icon (that Icon will magnify, meaning that you've selected it). Press the A Button to activate the currently selected Icon. Press the START Button again to remove the Command Cluster without activating anything. If a dialog is displayed, the START Button will close the dialog.

  • Left Shift Press to scroll the entire page UP without moving the Arrow.

  • Right Shift Press to scroll the entire page DOWN without moving the Arrow.


With the On-Screen Keyboard Displayed


When you move the cursor onto an entry field or URL bar and click the A Button on the controller, the Net Link browser will bring up an on-screen keyboard that can be used to enter information. This on-screen keyboard will also appear when you click on the entry fields on the e-mail to type an e-mail message.


When the on-screen keyboard is displayed, the Controller Buttons are the same as detailed above except where noted.

  • B Button Works as the Backspace Key Deletes the character behind the cursor. When combined with the Shift Key, deletes the previous word.

  • C Button Works as the Shift key

  • X Button Erases any input in the selected field and insert "http://www." When used with the C Button, inserts ".com/"

  • Y Button Inserts " " (space). When used with the C Button, it inserts "/".

  • Z Button Acts as Tab key, allowing you to move from Link to Link or field to field When used with the C Button, it acts as the Enter key

  • Left Shift Button Moves the cursor left in the selected field.

  • Right Shift Button Moves the cursor right in the selected field.


With the Bookmarks or Address Book Displayed


Buttons are the same as above except where noted.

  • B Button Closes the window, (the same as the START button).

  • C Button Clears the selected item.


On-Screen Keyboard Description

Special button

  • .com The end of many company domain names.

  • .net The end of many Internet Service Provider's domain names.

  • .org The end of many non-profit organization's domain names.

  • .gov The end of many government organization's domain names.

  • esc Erases all input in the selected entry.

  • enter Closes the on-screen keyboard and when in the URL bar, will put the keyboard away and start loading the new page.
